Student Management System (Java)

 Project Description
This is a console-based Student Management System developed using Java.
It performs basic CRUD operations using OOP concepts and ArrayList.

 Features
- Add Student
- Update Student
- Delete Student
- View All Students

Technologies Used
- Java
- OOP Concepts
- ArrayList

Project Structure
src
 ├── main
 ├── model
 ├── service

 Future Improvements
- Add Database (MySQL)
